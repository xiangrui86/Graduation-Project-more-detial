# 商品图片目录

本目录用于存储商品图片。

## 需要的图片文件

请在此目录下放置以下图片文件：

1. **football.jpg** - 足球商品图片 (推荐尺寸: 400x300px)
2. **basketball.jpg** - 篮球商品图片 (推荐尺寸: 400x300px)
3. **treadmill.jpg** - 跑步机商品图片 (推荐尺寸: 400x300px)
4. **badminton.jpg** - 羽毛球拍商品图片 (推荐尺寸: 400x300px)

## 使用方法

1. 将图片文件放到此目录
2. 重启后端服务
3. 图片会自动通过以下 URL 访问：
   - http://localhost:8080/images/football.jpg
   - http://localhost:8080/images/basketball.jpg
   - http://localhost:8080/images/treadmill.jpg
   - http://localhost:8080/images/badminton.jpg

## 图片来源

可以从以下网站下载免费的商品图片：

- **Unsplash**: https://unsplash.com
- **Pexels**: https://pexels.com
- **Pixabay**: https://pixabay.com

搜索关键词：

- 足球: "soccer ball"
- 篮球: "basketball"
- 跑步机: "treadmill" 或 "running machine"
- 羽毛球拍: "badminton racket"

## 快速替换图片

如果要更换某个图片，只需：

1. 删除旧的文件
2. 放入新的文件（保持相同名称）
3. 重启后端或清除浏览器缓存
